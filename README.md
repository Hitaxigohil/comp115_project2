Alien Invasion: Hitaxi Edition

This is a simple arcade-style shooting game made using Pygame.

Features:
Combo system: After 10 kills, auto-fire and red bullets are activated for a short time.
Particle effects: Aliens break into small glowing particles when hit.
Moving background: Stars scroll to create a space effect.
Kill tracker: Shows the total number of kills.

Technical Details:
Code is organized into multiple files.
Uses time functions to control power-ups.
Uses sprite groups for better performance.

Controls:
Arrow keys: Move
Space: Shoot
Q: Quit

About:
This project is an improved version of a basic Pygame shooter with added effects and gameplay features.
